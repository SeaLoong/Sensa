    快速操作

    快速上手

    从第一次启动到第一次设备动作，这里是一条最短路径。

    适合首次使用
    1. 确认 VRChat 已开启 OSC，默认向 127.0.0.1:9001 发送头像参数。
    2. 在"总览"页检查 Loop、TCode、Intiface、Recording 的当前状态。
    3. 在"配置"页填写信号路径、角色和设备连接配置。
    4. 在"测试"页用手动覆盖直接验证 OSR2 / SR6 / OSR6 的轴向和振动。
    5. 在"监控"页确认参数流、日志和录制链路都工作正常。

    提示：修改 OSC 端口后建议重启服务；其余大部分配置支持热更新。

    上手建议
    
              
    
                
    
                  
    1. 先跑通最短链路
    
                  
    优先验证：Loop 运行 → 手动测试有反应 → TCode / Intiface 连接成功 → 再接入 VRChat OSC。
    
                
    
                
    
                  
    2. 先做粗校准
    
                  
    信号先把 VrchatMin / VrchatMax 调到能覆盖输入范围，再微调 Dead Zone 和 Smoothing α。
    
                
    
                
    
                  
    3. 手机端也能用
    
                  
    移动端采用底部 Tab 导航，适合放在第二屏或远程桌面窗口中当控制面板。
    
                
    
                
    
                  
    4. 多主题 / i18n
    
                  
    右上角可切换 ZH / EN 界面语言以及配色主题（深蓝、AMOLED、紫色、翠绿、亮色）。

                Intiface 内嵌引擎：
